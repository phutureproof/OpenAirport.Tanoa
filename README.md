# OpenAirport

A cooperative sandbox mission for ArmA 3

## Features

- Airport job queueing system
- Shared money pool
- Full featured game saving system
- Shop where you can purchase new vehicles and personal items
- Passenger missions
- Parachute missions
- Medical missions
- Cargo mission
- ATC system
- Support for any mod that adds vehicles (CUPS, RHCS, etc)
- Support for ALiVE (not required)
- Support for local and dedicated server
