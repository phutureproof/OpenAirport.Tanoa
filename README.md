# OpenAirport

A cooperative sandbox mission for ArmA 3, take on the role of a pilot and earn money for your company and yourself by completing a variety of jobs.

## Features

- Airport job queueing system
- Shared airport funds for buying new vechiles and refueling / repairing
- Personal money to buy new items
- Full featured save system
- Shop where you can purchase new vehicles and personal items
- Multiple mission types
    - Passenger missions
    - Parachute missions
    - Medical missions
    - Cargo missions
- ATC system (Work in progress)
- Support for any mod that adds vehicles (CUPS, RHS, etc)
- Support for ALiVE (not required)
- Support for local and dedicated server
